#### Introducere

```
Acest proiect isi propune implementarea unei aplicatii in limbajul de programare Java care sa genereze date meteorologice despre o lista definita de orase.
```

#### Utilizare:

```
Aplicatia ofera utilizatorului o interfata grafica prin care poate interactiona cu functionalitatile. Aceasta interfata ofera utilizatorului posibilitatea de alegere a unui fisier in care sunt trecute datele despre orase, fisierul fiind ales prin introducerea caii intr-un textbox. Dupa alegerea acestuia, se poate incarca fisierul prin apasarea butonului "Load".
Dupa ce fisierul este incarcat, utilizatorului i se pun la dispozitie cele doua ComboBox-uri prin care isi poate alege tara din care vrea sa caute detalii despre un anumit oras, apoi, folosind al doilea ComboBox, isi poate alege orasul. Dupa ce a a facut acest lucru, poate folosi butonul "Search" pentru a trimite request-ul catre API-ul https://openweathermap.org ce va returna datele despre vreme. Aceste date vor fi afisate in interfata grafica.
```

Exemplu de fisier de intrare:

```
819827 Razvilka 55.591667 37.740833 RU
524901 Moscow 55.752220 37.615555 RU
2973393 Tarascon 43.805828 4.660280 FR
2986678 Ploufragan 48.491409 -2.794580 FR
```
